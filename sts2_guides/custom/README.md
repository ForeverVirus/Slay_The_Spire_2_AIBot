# Custom Knowledge

Place user-defined Slay the Spire 2 knowledge files here.

- Supported JSON files should match the built-in names such as `cards.json`, `relics.json`, `builds.json`.
- Supported Markdown files include `README.md`, character guide markdown, and other game-only notes.
- Files that contain code fences, URLs, or non-game/system prompt content are rejected by the mod validator.
- Custom files override built-in entries by matching `id` or `slug` where applicable.
